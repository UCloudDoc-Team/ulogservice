# 检索概述

## 基本概念

日志检索提供对日志数据进行过滤、搜索的功能，例如查询包含 `error` 的日志、关键字配置、统计日志条数，是日志服务中最常用的功能。

## 功能特性

### 基础功能

针对日志内指定的字段进行检索，指定字段满足检索条件时即可检索到该日志，例如添加相应的筛选条件，检索对应日志。

### 主题选择

1. 进入日志查询，选择查询主题。

![image](/images/switch_topic_1.png)

### 筛选条件

#### 添加筛选条件

不同字段类型支持的筛选条件如下：

| 文本选择      | 长整型选择   | 浮点型选择   |
| ------------ | ------------ | ------------ |
| 字段值为空    | 等于         | 等于         |
| 字段值不为空  | 不等于       | 不等于       |
| 包含         | 大于         | 大于         |
| 不包含       | 大于等于     | 大于等于     |
|              | 小于         | 小于         |
|              | 小于等于     | 小于等于     |
|              | 指定范围     | 指定范围     |
|              | 不在指定范围 | 不在指定范围 |

#### 日期时间范围选择

支持选择特定的时间范围进行检索。

![image](/images/search_topic_1.png)

### 检索结果

检索结果页面提供以下功能：

1. **日志统计图表展示**：直观展示检索结果的数据分布
2. **日志结果展示**：
   - 按日志时间排序
   - 日志滚动加载
